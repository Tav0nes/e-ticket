# e-ticket

An internal Incident/Ticket Management system. Personal learning project to mirror the kind of tooling used by enterprise digital-transformation teams.

## Stack
- **Frontend:** Angular 20+ (standalone components)
- **Backend:** NestJS (Node.js runtime)
- **Database:** PostgreSQL via TypeORM
- **Auth:** Keycloak (OIDC) — TBD Sprint 4
- **Local dev:** Docker Compose
